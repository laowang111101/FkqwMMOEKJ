# 招聘大数据可视化和推荐系统 python322

## 项目概述

招聘大数据可视化和推荐系统是一款基于Spark、Python和Vue的实时数据分析工具，旨在帮助用户更直观地理解和分析招聘信息。

## 技术栈

- **后端：**
  - Python 3.8
  - Flask框架
  - Spark
  - MySQL 5.7+
  - Hive
- **前端：**
  - Vue
  - iView
  - Echarts

## 功能模块

### 数据展示与分析

- **数据概览：**
  - 展示爬取数据，支持多条件查询，点击可跳转至相应网址。
- **城市平均薪资：**
  - 按工作城市分组，展示每个城市的平均薪资，使用Echarts进行热度图和柱状图展示。
- **技能词云：**
  - 分析公司招聘技能要求，通过词云图进行展示。
- **薪资统计：**
  - 通过行业、学历要求和工作地点分析薪资占比。
- **招聘要求分析：**
  - 以经验和学历为维度，分析各行业招聘要求占比。
- **公司信息分析：**
  - 对公司属性、性质和规模进行分析。
- **薪资预测：**
  - 使用工资中位数预测薪资。
- **职位推荐：**
  - 用户填写个人信息，系统根据用户信息模糊查找数据进行职位推荐。
- **真实性分析：**
  - 对职位薪资进行合理性比对。

### 系统管理

- **数据爬取：**
  - 输入关键字和页数进行在线数据获取，仅管理员权限。
- **爬取日志：**
  - 展示数据获取日志，仅管理员权限。
- **修改密码：**
  - 用户登录后可修改密码。
- **登录注册：**
  - 用户登录注册功能。
- **用户管理：**
  - 管理员管理用户信息。

## 系统角色

- **普通用户：**
  - 可查看数据分析结果，进行职位推荐和个人信息管理。
- **管理员：**
  - 具备普通用户所有功能，同时负责数据爬取、日志查看和用户管理。
